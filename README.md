# GigaQifa

GigaQifa — это интеллектуальный интерфейс для мультиагентной нейросетевой системы.

## Особенности
- **Мультиагентная архитектура**: Поддержка модулей Консультанта, Парсера и Валидатора.
- **Интеграция с Python**: Спроектировано для работы с FastAPI бэкендом.
- **Локализация**: Полная поддержка русского и китайского языков.
- **RAG (Retrieval-Augmented Generation)**: Загрузка файлов контекста для точных ответов.

## Разработка
1. Установите зависимости: `npm install`
2. Настройте `.env` (см. `.env.example`)
3. Запустите сервер разработки: `npm run dev`
